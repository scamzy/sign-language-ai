# AI Face Recognition + Sign Language Interpreter

## Features
- Face Detection
- Attention Monitoring
- Hand Gesture Recognition
- Sentence Translation
- Text-to-Speech Output
- Conversation History

## Technologies Used
- Python
- OpenCV
- MediaPipe
- pyttsx3

## How to Run
pip install opencv-python mediapipe pyttsx3  
python app.py

## Output
Real-time detection with gesture-to-sentence translation and voice output.
